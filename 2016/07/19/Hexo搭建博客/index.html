hexo博客的搭建

发表于 2016-07-19

Hexo搭建博客

环境的安装

安装 node

选择一个版本下载，下载完一路确定安装即可， 自带npm 。

安装 git

下载一路安装即可。

hexo的安装

新建文件夹

在桌面或某个磁盘新建一个空文件夹，存放hexo，后续操作都在这个文件夹操作

打开 Git Bash

在新建的文件夹内右键鼠标 打开 Git Bash

npm install hexo-cli -g //输入完回车即可，用于全局安装hexo

hexo的安装

hexo init //初始化hexo，这个过程会下一些文件到目录下，下载完成即可。

安装hexo依赖

npm install

安装过程中报错

如果安装过程中有类似ERROR Deployer not found : github的错误。

输入命令npm install hexo-deployer-git --save即可。

hexo命令

hexo g 初始化hexo修改的内容

hexo s 访问到本地的hexo博客(到这一步可以看到自己搭的博客了-.-)

在浏览器上输入 http://localhost:4000/ 可以看到自己本地的博客

hexo d 提交到github上输完命令后一般会输入用户名密码，正确后会提交到github上

推送到github

  1. 注册github填写邮箱和密码，注册有邮箱会收到验证信息，确定即可

  2. 新建New repository

  3. Repository name 为 : 创建github的名称 + .github.io 比如 ： z-weiH.github.io

  4. 创建完毕后 本地的代码 hexo d 提交到 gitbug 上 ， 页面直接访问 刚才创建的文件名即可 在线博客

next主题说明

站点配置文件

主题配置文件

  1. 下载主题到/themes文件夹下
  2. 站点配置文件中theme : 这里改为下载主题的名称
  3. 主题的配置项在/thems/主题名/_config.yml下根据主题配置文档，进行相关配置

其他-关于博客文章的删除

  1. 再把hexo文件根目录找到db.json把他干了
  2. 执行 hexo clear
  3. hexo d 推送到github上可以看到相关文章已经删除
